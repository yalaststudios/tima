Privacy Policy

Last updated: March 23, 2026

1. Introduction

Tima is a course scheduling app by Yalast Studios. This Privacy Policy explains how we handle data when you install and use the app.

2. Data collection

  • Tima does not collect or store personal data (name, email, phone, profile, etc.).
  • No analytics, crash reporting, or device identifier tracking is used.
  • No data is sent to remote servers by default.

3. Local storage

Course schedules, events, and app settings are stored locally on the device (localStorage/app storage). Data remains on the device and is not synced to cloud services.

4. Permissions

Tima may request permission for local storage/file access for saving and exporting/importing schedule data. No location, camera, microphone, contacts, or calendar permissions are used.

5. Third-party sharing

Tima does not share user data with third parties. No ads, no analytics providers, and no external APIs are used for personal data.

6. Security

We use standard local storage practices. Users are advised to back up data manually using export features if they wish.

7. Children

This app is not intended for children under 13. No personal data collection is performed.

8. User rights

Since no personal data is collected, there is no personal data to access or remove from any server.

9. Contact

For privacy questions, contact: yalaststudios@gmail.com

10. Changes to this policy

Any change in data collection or cloud usage will be updated here with a new effective date.
