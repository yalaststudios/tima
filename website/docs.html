Documentation

Helpful steps for using Tima effectively.

These guides are written for real use, not for perfect conditions. Start small, set up the parts that matter most, and improve your routine as you go.

Setup Schedule planning Events and deadlines Notifications Backups Helpful habits Troubleshooting

Setup

Recommended first-time setup

  1. Complete onboarding and allow notifications if you want reminders.
  2. Add your weekly classes first so the rest of the app has context.
  3. Choose whether you prefer 12-hour or 24-hour time in Settings.
  4. Open the Events page and add the deadlines you already know.
  5. Visit Backup in Settings and export your first file after setup.

Schedule planning

How to make the schedule page work better

  • Use edit mode when you are actively building or changing your timetable.
  • Keep venue names consistent so your schedule stays easier to scan.
  • Use Auto-scroll to today if you mostly care about what is next, not the full week at once.
  • Search is best for quickly checking one course without scrolling through every day.

Events and deadlines

What belongs in Events

Tima supports tests, presentations, assignments, exams, and other date-based items. This page works best when you add important academic dates as soon as you hear about them.

  • Add deadlines early, even if the exact hour is not final yet.
  • Group your attention by month so busy periods are visible sooner.
  • Use the relative status labels to check what is today, tomorrow, or already passed.

Notifications

How reminders work

  • Class reminders are sent 30 minutes before saved classes.
  • Event reminders are scheduled 2 days before, 1 day before, and on the event date.
  • Course start alerts can notify you right when a course begins.

Important Android note

On Android, sound and vibration behavior is controlled by notification categories. Open the notification settings inside Tima if you want to adjust sound, vibration, or pop-on-screen behavior for each category.

If reminders seem late

  • Confirm that notifications are enabled for the app.
  • Reduce battery restrictions or set the app to a less restricted battery mode.
  • Double-check your class and event times for accuracy.

Backups

What backup export includes

Tima exports schedules, events, and saved suggestions into a JSON backup file. The app uses Android's file picker so you can choose where the backup is saved.

Best backup habit

  • Export a backup after major changes.
  • Keep at least one backup in a location you trust.
  • Import only files that came from your own Tima setup.

Helpful habits

A steady routine beats a complicated one

  • Review tomorrow's classes before you sleep.
  • Check Events once at the start of the week and once midweek.
  • Use Free Time to spot your lightest day before you commit to extra tasks.
  • Keep your reminders supportive, not overwhelming.

Troubleshooting

Common questions

Why do I not hear reminder sounds?

Check the Android notification category settings for that reminder type.

Why is the app asking about file access during backup actions?

Tima uses Android's document picker for import and export so you can choose the location yourself.

Why are reminders still unreliable on some phones?

Some devices delay background work when battery optimization is aggressive. Reducing restrictions usually helps.
